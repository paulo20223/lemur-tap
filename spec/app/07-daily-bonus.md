# Ежедневный бонус

## Стрик-таблица

| День | Награда |
| --- | --- |
| 1 | 1000 |
| 2 | 2500 |
| 3 | 5000 |
| 4 | 10000 |
| 5 | 20000 |
| 6 | 35000 |
| 7+ | 50000 |

## Правила

- Клейм доступен 1 раз в UTC-сутки.
- День 7+ повторяется, пока стрик жив.
- Пропуск календарного дня → сброс стрика на День 1.

## Идемпотентность

- Уникальность по паре `(userId, UTC-дата)`.
- Idempotency-ключ в Redis на момент клейма.
- Повторный клейм в те же сутки невозможен.
