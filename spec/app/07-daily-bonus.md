# Ежедневный бонус

## Стрик-таблица

Дейлик — **ретеншн-добавка, а не основной доход**: стабильная награда ≈ одной хорошей сессии. Подробнее о ребалансе и причинах — [12 — Прогрессия и баланс](./12-progression.md#6-ребаланс-дейлика-важно).

| День | Награда |
| --- | --- |
| 1 | 500 |
| 2 | 750 |
| 3 | 1000 |
| 4 | 1500 |
| 5 | 2000 |
| 6 | 3000 |
| 7+ | 5000 |

> Прежняя таблица (1000…50000) давала 50k/день — в ~6 раз больше дневного гринда и ломала принцип «надо поработать». Снижено осознанно.

## Правила

- Клейм доступен 1 раз в UTC-сутки.
- День 7+ повторяется, пока стрик жив.
- Пропуск календарного дня → сброс стрика на День 1.

## Идемпотентность

- Уникальность по паре `(userId, UTC-дата)`.
- Idempotency-ключ в Redis на момент клейма.
- Повторный клейм в те же сутки невозможен.
