# Рефералы

## Ссылка

У каждого игрока есть `referralCode`. Приглашение происходит через Telegram deep link: `startapp=ref_<code>`. При первом запуске приглашённого создаётся запись `Referral` (с защитой от самоприглашения и повторов).

## Бонусы

| Событие | Реферер | Реферал |
| --- | --- | --- |
| Регистрация по ссылке | +75 | +50 |
| Реферал — Telegram Premium | +150 | +150 |
| Пассив | отключён (`referralPassiveRate = 0`) | — |

## Реализация

Пассивная доля от заработка реферала **отключена** (`referralPassiveRate = 0` в `GameConfig`): при нулевой ставке `mintReferralPassive` сразу возвращает 0 и строк `refSource='passive'` больше не пишется. Историческое поле и код ветки сохранены, чтобы пассив можно было вернуть правкой live `GameConfig` без редеплоя. Разовые бонусы (`join` / `premium`) по-прежнему записываются в леджер реферера/реферала с типом `'referral'`. Подробнее о валютах и леджере — [04 — Экономика](./04-economy.md).
