# Change Spec — Купон-буст (одноразовый расходник)

Статус: Implemented · 2026-06-15 · затрагивает [06](../06-coupon-game.md), [03](../03-data-model.md), [04](../04-economy.md), [orpc/05](../../orpc/05-procedures.md)

Change-спека для одноразового буста купонной мини-игры, живущего в разделе **Награды → Бусты**.

---

## 1. Мотивация

Раздел «Бусты» (вкладка Rewards) до этого содержал только перманентные апгрейды. Нужен **расходник-крючок**: быстрый способ сыграть ещё раз, не дожидаясь регена энергии (час на раунд).

## 2. Механика

Покупка процедурой `coupon.boost` за `coupon_boost_price` монет (по умолчанию 100 ≈ типичная выплата за раунд; антифрод-кап `coupon_max_coins` ~3000 как цена был бы экономически абсурден) разово **доливает энергию на одну попытку**: `+coupon_boost_energy_grant` (= `coupon_session_cost`), зажат по `max_energy`. Бар держит ровно один раунд, поэтому опустошённый бар доливается до полного.

Покупка атомарна (списание монет + долив энергии в одной транзакции); сама игра/награда раунда не меняется. Никакого per-user флага не вводится.

> История: первая итерация добавляла ещё и взвод бонус-дропа (1/20 на +50 монет сверх кэпа) с флагом `User.couponBoostActive`. Механика отменена — буст оставлен чисто как «монеты → энергия на попытку».

## 3. Дельта к данным/конфигу/API

- **Data** ([03](../03-data-model.md)): схема `User` не меняется; ledger-тип `coupon_boost` (списание за покупку).
- **Config** ([04](../04-economy.md)): `coupon_boost_price=100`, `coupon_boost_energy_grant=500`. Версия `GameConfig` 6 → 7.
- **API** ([orpc/05](../../orpc/05-procedures.md)): новая процедура `coupon.boost` (input —, output `{ coins, energy, energyUpdatedAt }`, authed, лимит `coupon`, ошибка `INSUFFICIENT_COINS`).
- **Shared**: `CouponBoostResponseSchema` в DTO; новые config-поля в `GameConfigSchema`.

## 4. Чек-лист внедрения

- [x] `shared`: enum `coupon_boost`, 2 config-поля + bump версии, DTO, контракт.
- [x] `api`: `CouponService.boost()` (списание монет + долив энергии), роутер.
- [x] `webapp`: `apiClient.couponBoost()`, блок буста в панели Бустов, i18n (ru/en).
