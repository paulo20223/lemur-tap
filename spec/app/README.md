## Глоссарий

- **энергия** — расходуемый ресурс: тратится на старт раунда мини-игры, регенерируется со временем.
- **монеты** — основная игровая валюта (не реальные деньги, без блокчейна).
- **coupon_mult** — множитель награды за раунд мини-игры; повышается веткой апгрейда.
- **стрик** — серия дней подряд с получением ежедневного бонуса.
- **харвест** — объём энергии, реально снятый за день (≈ число сессий × maxEnergy).
- **стейкинг** — офлайн-движок дохода: монеты копят пассив в хранилище (ёмкость = ветка Vault), забираемый ежедневным клеймом.
- **Vault** — ветка апгрейда, поднимающая ёмкость хранилища стейкинга (дневной потолок дохода).
- **реферал** — приглашённый пользователь, привязанный через deep link `startapp`[^startapp].
- **initData** — подписанные данные Telegram, проверяемые на сервере HMAC-SHA256[^initdata].

## Сноски

[^tma]: Telegram Mini Apps — https://core.telegram.org/bots/webapps
[^initdata]: Валидация initData (HMAC-SHA256) — https://core.telegram.org/bots/webapps#validating-data-received-via-the-mini-app
[^startapp]: Deep links startapp — https://core.telegram.org/api/links#main-mini-app-links
