## Глоссарий

- **энергия** — расходуемый ресурс тапа, регенерируется со временем.
- **монеты** — основная игровая валюта (не реальные деньги, без блокчейна).
- **tap_power** — количество монет за один тап; повышается апгрейдами.
- **стрик** — серия дней подряд с получением ежедневного бонуса.
- **харвест** — объём энергии, реально снятый за день (≈ число сессий × maxEnergy).
- **стейкинг** — заморозка монет под процент с начислением дохода.
- **реферал** — приглашённый пользователь, привязанный через deep link `startapp`[^startapp].
- **initData** — подписанные данные Telegram, проверяемые на сервере HMAC-SHA256[^initdata].

## Сноски
